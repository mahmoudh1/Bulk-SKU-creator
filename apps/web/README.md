# Bulk-SKU-Creator Web

Frontend app for Bulk-SKU-Creator.

## Local development

- Install deps (from repo root):

```bash
npm install
```

- Run the web app (from repo root):

```bash
npm run dev:web
```

- Quality checks (from repo root):

```bash
npm run test:web
npm run lint:web
```

## Environment

Set `VITE_CLERK_PUBLISHABLE_KEY` in `.env.local` (see `.env.example`).
